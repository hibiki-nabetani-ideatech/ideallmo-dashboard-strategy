# IDEA LLMO Dashboard / Strategy

ペルソナ起点の戦略ダッシュボード（GitHub Pages 公開用・単一自己完結HTML）。

- 公開範囲: URLを知っている人のみ（`<meta robots=noindex>` + `robots.txt` で検索エンジン除外）
- 構成: ① ペルソナA / ② ペルソナB / ③ ペルソナC（各：強みの言語化とRTB / LLMの推薦基準と推薦企業 / AI認知マッピング / 施策リスト）＋ ④ IDEAナレッジ（主要AIニュース / 実施施策・施策結果 / IDEAコンサルティング）
- `index.html` 単体で完結（データ・画像 base64 を内包）。`_pipeline/` は生成元（任意）。
